## Technologies Used
- Java
- IntelliJ IDEA / Eclipse IDE
- Maven

## Objective
To understand the fundamentals of Java programming, software design principles, and problem-solving by implementing hands-on exercises using industry-standard tools.
## How to Run
1. Clone or download this repository.
2. Open the required project in IntelliJ IDEA or Eclipse.
3. Wait for the project to load and download dependencies (if using Maven).
4. Build the project.
5. Run the Java file containing the `main()` method.
6. View the output in the console.
## Learning Outcomes
- Learned Object-Oriented Programming concepts.
- Implemented common Design Patterns.
- Improved problem-solving using Data Structures & Algorithms.
- Practiced PL/SQL programming.
- Understood Unit Testing and Logging basics.
