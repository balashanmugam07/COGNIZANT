## Objective
To learn how to build RESTful web applications using Spring Boot, manage dependencies with Maven, perform database operations using Spring Data JPA, and follow a layered architecture for developing enterprise applications.
## How to Run
1. Clone or download the repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Wait for Maven to download all required dependencies.
4. Configure the database in `application.properties` (if required).
5. Build the project.
6. Run the Spring Boot main application class.
7. Open Postman or a web browser to test the REST API endpoints.

## Learning Outcomes
- Developed RESTful APIs using Spring Boot.
- Implemented CRUD operations with Spring Data JPA.
- Understood layered architecture.
- Managed project dependencies using Maven.
- Tested APIs using Postman.
