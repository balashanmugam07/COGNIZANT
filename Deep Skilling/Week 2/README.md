## Deepskilling (Week 2)
## Skills
- Spring Core
- Spring Boot
- Spring Data JPA
- Hibernate ORM
- Maven
- REST API Development

## Technologies Used
- Java
- IntelliJ IDEA / Eclipse IDE
- Maven
- Spring Framework
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## Objective
To learn the fundamentals of Spring Framework, build Maven-based applications, connect with databases using Spring Data JPA and Hibernate, and develop RESTful web applications.
## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Wait for Maven to download all dependencies.
4. Configure the database details in `application.properties` (if required).
5. Build the project using Maven.
6. Run the main Spring Boot application class.
7. Open the browser or Postman to test the application.
## Learning Outcomes
- Understood Spring Core concepts.
- Built Spring Boot applications.
- Connected applications with MySQL.
- Performed database operations using Spring Data JPA and Hibernate.
- Developed and tested REST APIs.
