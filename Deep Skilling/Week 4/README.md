## How to Run
Clone or download the repository.

Open the project in IntelliJ IDEA or Eclipse.

Wait for Maven to download all required dependencies.

Start the Eureka Server.

Run the required microservices.

Start the API Gateway.

Test the APIs using Postman or a web browser.

## Objective
To learn code quality analysis using SonarQube, develop microservices with Spring Boot 3, implement Spring Cloud components, and build scalable enterprise applications.
